Интеграция с Next.js

Добавьте этот код в ваш Next.js проект для отслеживания аналитики:

Замените ЗАМЕНИТЕ_НА_ВАШ_WEBSITE_ID на ID вашего сайта из Umami дашборда.

Как получить Website ID

  1. Откройте https://analytics.borisovai.ru
  2. Войдите в свой аккаунт (используйте credentials из Management UI)
  3. В левой навигации выберите нужный сайт (или создайте новый через "Add website")
  4. На странице сайта найдите секцию "Tracking code"
  5. Скопируйте значение атрибута data-website-id из предложенного кода
  6. Вставьте его в свой Next.js проект в карту выше
