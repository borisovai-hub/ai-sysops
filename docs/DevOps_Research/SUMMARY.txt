ИНТЕГРАЦИЯ С СУЩЕСТВУЮЩИМ ПРОЕКТОМ
================================================================================

Существующие компоненты borisovai-admin:
  - Management UI (Express.js)
  - Traefik (reverse proxy)
  - GitLab CE (CI/CD)
  - Authelia (SSO)
  - frp (tunneling)
  - Umami (analytics)

Подход интеграции:
  - Extension: новые модули в Management UI
  - Gradual Migration: постепенный переход на новую систему
  - API-первый подход: работаем через existing APIs
